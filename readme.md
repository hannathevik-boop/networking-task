## Reflection

In this module, I learned how the internet is structured using the TCP/IP model and how communication between clients and servers works. The socket API made it clear how programs can send and receive data over networks. I now also better understand how browsers use HTTP to request data from web servers.

What surprised me most was how simple it is to resolve a hostname into an IP address using Python. I expected it to be more complex.

One topic I would like to explore further is how HTTP requests are structured and how web servers handle multiple clients at the same time.
